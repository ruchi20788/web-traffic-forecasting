# 📊 Web Traffic Forecasting System

## 🚀 Overview

A Flask-based web application that predicts website traffic using Machine Learning and Time Series models.

## 🧠 Models Used

* Random Forest
* SARIMAX
* LSTM

## 📌 Features

* Login system
* Website search & selection
* Forecast future traffic
* Backtesting metrics
* CSV export

## 🛠️ Tech Stack

Python, Flask, Pandas, NumPy, Scikit-learn, Statsmodels, TensorFlow

## 📂 Structure

```
app.py
forecast_utils.py
templates/
static/
data/
requirements.txt
```

## ▶️ Run

```
pip install -r requirements.txt
python app.py
```
